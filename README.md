# CodeIn

A premium single-page React portfolio website for a freelance digital collective based in Bangalore. The project features an editorial-style design, glassmorphism effects, smooth scrolling, interactive cards, and a contact form — all implemented in a single React component using inline styles.

---

## Overview

CodeIn is a modern portfolio website designed to showcase a digital agency or freelance collective. It includes:

- Editorial hero section with premium typography
- Team member showcase cards
- Project and impact portfolio section
- Contact form with polished UI
- Light and dark theme toggle
- Smooth scrolling navigation
- Responsive design for desktop, tablet, and mobile
- Single-file React implementation with inline styles

---

## Features

### Hero Section
- Large Playfair Display heading
- Gradient accent styling
- Call-to-action button
- Responsive editorial layout

### Team Showcase
- Interactive profile cards
- External portfolio links
- Hover animations

### Projects & Impact
- Case study cards
- Impact badges
- Technology tags

### Contact Form
- Controlled React state
- Name, email, phone, and project description fields
- Frontend-only submission handling

### Theme Toggle
- Dark and light mode
- Dynamic color tokens

---

## Technology Stack

- Google Fonts (Playfair Display, DM Sans)
